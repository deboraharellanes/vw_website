Preparing to Install the Engine

Make sure everything is in place before installing the engine. Handle all the parts with extreme caution to avoid any damage.

  1. Center the clutch plate
  2. Check the release bearing and release plate for wear
  3. Lubricate the starter bush with lithium grease
  4. Add universal grease to the the needle bearing
  5. Lubricate the main drive shaft splines with molydenum-disulphide powder
  6. Clean the mating surfaces of the engine and transmission

Installing the Engine

  1. Lift the engine into position
